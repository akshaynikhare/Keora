## Getting Started

### What is Keora?

Keora is a family tree platform that helps you:
- Build and visualize your family tree
- Connect with relatives through verified connections
- Share your family history and heritage
- Collaborate with family members to build a comprehensive tree

### System Requirements

- **Web Browser:** Chrome, Firefox, Safari, or Edge (latest version)
- **Mobile:** iOS 13+ or Android 8+
- **Internet Connection:** Required for all features

## Creating Your Account

### Sign Up Process

1. **Visit Keora**
   - Go to https://keora.com
   - Click "Sign Up" button

2. **Enter Your Information**
   - Full Name
   - Email Address (will be verified)
   - Mobile Number (with country code, e.g., +1234567890)
   - Password (minimum 8 characters)
   - Date of Birth (optional but recommended)

3. **Password Requirements**
   - Minimum 8 characters
   - Recommended: Mix of uppercase, lowercase, numbers, and symbols
   - Password strength indicator will guide you

4. **Verify Your Account**
   - Check your email for a 6-digit code
   - Check WhatsApp for a 6-digit code
   - Enter both codes on the verification page
   - Your account is now active!

### Alternative Sign Up

You can also sign up using:
- **Google Account** - Quick sign up with your Gmail
- **Facebook Account** - Use your Facebook profile

## Building Your Family Tree

### Adding Your First Family Member

1. **Go to Dashboard**
   - Click "Add Family Member" card

2. **Enter Details**
   - Name (required)
   - Photo (optional, recommended)
   - Date of Birth
   - Gender
   - Location
   - Bio (short description)

3. **Privacy Level**
   - **Private:** Only you can see
   - **Family:** Connected family members can see
   - **Public:** Anyone can see in search results

4. **Mark as Primary**
   - Check "Primary Profile" for yourself
   - This becomes your main profile in the tree

### Adding More Family Members

**Parents:**
1. Select yourself in the tree
2. Click "Add Parent"
3. Choose: Father or Mother
4. Fill in details
5. Save

**Siblings:**
1. Select yourself or a parent
2. Click "Add Sibling"
3. Fill in details
4. Connection is created automatically

**Children:**
1. Select yourself or spouse
2. Click "Add Child"
3. Fill in details
4. Both parents can be linked if applicable

**Spouse:**
1. Select yourself
2. Click "Add Spouse"
3. Fill in details
4. Marriage relationship is created

### Relationship Types

Keora supports:
- **Parent/Child** - Direct lineage
- **Spouse** - Marriage connections
- **Sibling** - Brothers and sisters
- **Extended Family** - Coming soon!

## Connecting with Family

### Finding Family Members

1. **Search Feature**
   - Click "Find Family" on dashboard
   - Enter name, email, or location
   - Filter by age range or location

2. **View Public Trees**
   - Browse public family trees
   - Look for familiar names
   - Check connection suggestions

### Sending Connection Requests

1. **Find the Person**
   - Use search to find relatives

2. **Send Request**
   - Click "Connect" button
   - Select relationship type (e.g., "They are my father")
   - Add a personal message (optional but recommended)
   - Click "Send Request"

3. **What Happens Next**
   - They receive a notification
   - They review your profile
   - They approve or decline
   - If approved, your trees merge!

### Managing Requests

**Received Requests:**
- Go to "Notifications" icon (bell)
- Review connection requests
- View the sender's profile
- Choose: Approve, Decline, or Block

**Sent Requests:**
- View status: Pending, Approved, or Declined
- Withdraw requests if needed
- Cannot send duplicate requests

### Merging Trees

When a connection is approved:
- Both trees are linked at the connection point
- You can now see their family members (based on privacy settings)
- Collaborative editing is possible
- Changes sync in real-time

## Privacy Settings

### Your Tree Privacy

**Access Settings:**
- Go to Settings > Privacy
- Choose tree visibility:
  - **Private:** Only you see your tree
  - **Family:** Only connected family members
  - **Public:** Anyone can search and view

### Individual Member Privacy

Each family member can have their own privacy level:
- **Private:** Hidden from everyone except you
- **Family:** Visible to connected family
- **Public:** Searchable and visible to all

### What Information is Shared?

**Always Visible (when public):**
- Name
- Profile picture
- Gender

**Controlled by Settings:**
- Date of Birth (can hide year)
- Location (can hide or show city only)
- Bio and additional details

### Search Visibility

- **Allow Search:** Let others find you in search results
- **Disable Search:** You won't appear in searches
- Note: Direct links to your profile still work

## Managing Your Profile

### Edit Your Information

1. **Go to Profile Settings**
   - Click your name/avatar
   - Select "Profile Settings"

2. **Update Fields**
   - Name, email, mobile
   - Profile picture
   - Bio, location
   - Date of birth

3. **Change Password**
   - Go to "Security" tab
   - Enter current password
   - Enter new password (twice)
   - Save changes

### Profile Picture

**Recommended:**
- Clear, front-facing photo
- Good lighting
- Minimum 400x400 pixels
- Max file size: 5MB
- Formats: JPG, PNG, WEBP

### Managing Sessions

**Active Sessions:**
- View all devices where you're logged in
- See device type, location, last active
- Logout from individual devices
- "Logout All" for security

## Tips & Best Practices

### Building an Accurate Tree

1. **Start with what you know**
   - Add immediate family first
   - Expand to grandparents
   - Then add extended family

2. **Verify information**
   - Double-check dates and names
   - Use official documents when possible
   - Ask family members to verify

3. **Add details**
   - Photos make trees more engaging
   - Bios preserve family stories
   - Locations show family migration

### Engaging with Family

1. **Send thoughtful requests**
   - Include a personal message
   - Explain your relationship
   - Be patient waiting for approval

2. **Respect privacy**
   - Don't share sensitive information publicly
   - Ask before adding someone else's details
   - Honor privacy preferences

3. **Collaborate**
   - Work together with family
   - Share research and photos
   - Discuss and resolve conflicts

### Keeping Your Account Secure

1. **Strong password**
   - Use a unique password
   - Enable password manager
   - Change regularly

2. **Email and mobile verification**
   - Keep both verified
   - Update if they change
   - These help recover your account

3. **Be cautious**
   - Don't share your password
   - Logout on shared devices
   - Review active sessions regularly

## FAQ

### Account & Login

**Q: I forgot my password. How do I reset it?**

A: Click "Forgot Password" on the login page. Enter your email or mobile number. You'll receive a reset code via email and WhatsApp. Use this code to set a new password.

**Q: Can I change my email address?**

A: Yes! Go to Profile Settings > Account > Change Email. You'll need to verify the new email address.

**Q: Why do I need to verify both email and mobile?**

A: This ensures account security and helps with password recovery. It also enables important notifications via both channels.

### Family Tree

**Q: How many family members can I add?**

A: There's no limit! Build as large a tree as you need.

**Q: Can I delete a family member?**

A: Yes, but be careful. Deleting a member removes all their connections. Consider making them private instead.

**Q: What if I make a mistake?**

A: You can edit any family member's information anytime. Click the member and select "Edit."

**Q: Can multiple people edit the same person?**

A: If you're connected, yes! Changes sync automatically. The tree shows who made the last edit.

### Connections

**Q: Someone sent me a connection request. How do I know if it's legitimate?**

A: Check their profile, mutual connections, and the relationship they claim. If unsure, message them first or decline.

**Q: Can I disconnect from someone?**

A: Yes. Go to the connection and select "Remove Connection." This doesn't delete data, just the link.

**Q: What happens if a connection request is declined?**

A: Nothing happens to your tree. You can't send another request to the same person for 30 days.

### Privacy

**Q: Who can see my tree?**

A: It depends on your privacy settings:
- Private: Only you
- Family: Only connected family members
- Public: Anyone

**Q: Can I hide specific family members?**

A: Yes! Set their privacy level to "Private" and only you will see them.

**Q: How do I make my tree public?**

A: Settings > Privacy > Tree Visibility > Select "Public"

### Technical Issues

**Q: The app is slow/not loading. What should I do?**

A: Try these steps:
1. Refresh the page (F5 or Cmd+R)
2. Clear browser cache
3. Try a different browser
4. Check your internet connection
5. Contact support if issue persists

**Q: I can't upload a photo. Why?**

A: Check:
- File size is under 5MB
- Format is JPG, PNG, or WEBP
- You have a stable internet connection
- Try a different image

**Q: My verification code expired. Can I get a new one?**

A: Yes! Click "Resend Code" on the verification page. You can request a new code every 60 seconds.

## Troubleshooting

### Can't Log In

**Issue:** "Invalid credentials" error

**Solutions:**
1. Check email/mobile spelling
2. Verify password (case-sensitive)
3. Try "Forgot Password" if needed
4. Clear browser cookies
5. Check if account is verified

### Not Receiving Emails

**Issue:** Verification or reset emails not arriving

**Solutions:**
1. Check spam/junk folder
2. Verify email address is correct
3. Add noreply@keora.com to contacts
4. Try resending after 5 minutes
5. Use mobile verification instead

### Connection Request Issues

**Issue:** Can't send connection request

**Solutions:**
1. Check if already connected
2. Check if request is pending
3. Verify target user allows connections
4. Try again in 24 hours if recently declined

### Tree Not Loading

**Issue:** Family tree visualization won't display

**Solutions:**
1. Refresh the page
2. Clear browser cache
3. Try different browser
4. Disable browser extensions
5. Check internet speed (minimum 1 Mbps)

### Profile Picture Won't Upload

**Issue:** Image upload fails

**Solutions:**
1. Reduce file size (max 5MB)
2. Convert to JPG or PNG
3. Check internet connection
4. Try different image
5. Use mobile app instead
